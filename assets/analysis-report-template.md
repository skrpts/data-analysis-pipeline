**Formatting Notes:**
- Report exact p-values to three decimal places (e.g., p = .023), except when p < .001
- Italicise statistical symbols: *t*, *F*, *p*, *r*, *d*, *M*, *SD*, *n*, *N*
- Use APA 7th edition table and figure formatting unless the target journal specifies otherwise
- Number tables and figures sequentially in the order they are mentioned in the text
- Place each table and figure as close as possible to its first mention, or in an appendix if required by the journal
